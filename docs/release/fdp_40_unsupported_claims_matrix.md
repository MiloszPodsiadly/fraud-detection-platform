# FDP-40 Unsupported Claims Matrix

| Evidence | Unsupported Claim |
| --- | --- |
| signed image | signed image does not mean external finality |
| signed image | signed image does not mean business correctness |
| signed image | signed image does not mean distributed ACID |
| attestation | attestation does not mean legal notarization |
| approval gate | approval gate does not mean operational correctness |
| rollback plan | rollback plan does not mean zero-downtime guarantee |
| release image digest | release image digest does not mean Kafka exactly-once |
| environment approval | environment approval does not mean bank certification |
| READY_FOR_ENABLEMENT_REVIEW | READY_FOR_ENABLEMENT_REVIEW does not mean PRODUCTION_ENABLED |
| fixture proof | fixture proof does not mean production image proof |

FDP-40 prevents future overinterpretation of signed provenance.
